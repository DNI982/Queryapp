# **App Name**: DataWise AI

## Core Features:

- Conexión a Bases de Datos: Conectar a múltiples bases de datos (PostgreSQL, MongoDB, MariaDB, Oracle, etc.) simultáneamente.
- Reconocimiento de Bases de Datos con IA: Implementar un agente de IA (en Python) que reconozca automáticamente los esquemas de bases de datos y use el razonamiento para incorporar esta información como herramienta durante la ejecución de consultas. El modelo necesitará el esquema de la base de datos o scripts de generación de consultas para identificar la estructura de la base de datos. Usa herramienta.
- Lenguaje Natural a SQL: Convertir las consultas en lenguaje natural del usuario (por ejemplo, 'muéstrame las ventas de los últimos tres meses ordenadas de mayor a menor precio') en consultas SQL.
- Generación y Exportación de Informes: Permitir a los usuarios descargar los informes generados en formatos Excel o PDF.
- Autenticación de Usuario: Implementar un sistema de inicio de sesión seguro para usuarios.
- Módulos del Panel de Control: Implementar módulos principales (usando Angular o Node.js para la UI): Consultas, Fuentes de Datos, Historial, Analíticas y Configuración, según la estructura HTML proporcionada.

## Style Guidelines:

- Color primario: Azul Eléctrico (#7DF9FF) para representar datos e inteligencia.
- Color de fondo: Azul marino oscuro (#191970), una versión desaturada de Azul Eléctrico, para una sensación sofisticada.
- Color de acento: Azul cielo (#87CEEB) como un tono análogo que proporciona un buen contraste y atrae la atención del usuario.
- Fuente para el cuerpo y los encabezados: 'Inter' sans-serif para un diseño moderno y neutral.
- Fuente para el código: 'Source Code Pro' para mostrar fragmentos de código.
- Usar iconos basados en datos y analíticos para diferentes tipos de bases de datos.
- Transiciones y animaciones sutiles al cargar datos y ejecutar consultas.
